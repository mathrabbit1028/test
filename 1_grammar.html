CSS 적용하기

이 페이지는 CSS를 적용하는 두 가지 방법에 대한 내용을 다룹니다.

태그를 이용하는 방법

style 태그를 이용해 CSS를 적용할 수 있습니다. style 태그 안에 CSS 코드를 넣으면 됩니다.
태그를 이용해 넣은 CSS 코드는 전체 body 안에 적용됩니다.

속성을 이용하는 방법

style 속성을 이용해 CSS를 적용할 수 있습니다. style 속성에 CSS 코드를 넣으면 됩니다.
속성을 이용해 넣은 CSS 코드는 style 속성을 넣은 태그에만 적용됩니다.

문법

태그: <style>(CSS code)</style>
속성: <(TAG name) style="(CSS code)">
