CSS 선택자

CSS에서 선택자는 디자인 요소를 변경할 대상을 말합니다.
CSS 선택자는 여러 종류가 있으며, 선택자는 전체 body에 영향을 주는 style 태그에서 사용됩니다.
하나의 선택자에 대해 여러 CSS 속성을 변경할 수 있으며, 각각의 속성은 동일하게 세미콜론(;)으로 구분되어야 합니다.

선택자 1: 태그

HTML 태그 자체가 선택자의 역할을 할 수 있습니다.
만약 특정 HTML 태그를 선택자로 사용하면 그 태그에 해당되는 모든 내용이 CSS의 영향을 받습니다.

선택자 2: class

class는 내용을 분류하여 묶기 위해 사용되는 인자로, HTML 속성을 통해 지정할 수 있습니다.
만약 특정 class를 선택자로 사용하면 그 클래스에 해당되는 모든 내용이 CSS의 영향을 받습니다.

선택자 3: id

id은 고유 이름을 지정하기 위해 사용되는 인자로, HTML 속성을 통해 지정할 수 있습니다.
만약 특정 id를 선택자로 사용하면 그 id에 해당되는 모든 내용이 CSS의 영향을 받습니다.

우선순위

특정 부분이 선택자에서 중복이 있을 경우 id, class, 태그 순으로 우선 처리됩니다. (우선 순위)
만약, 선택자의 종류가 같다면, body와 가까운 코드(나중에 실행되는 코드)로 처리됩니다.
구체적인 예시는 아래 "예시"를 참고하세요.

중복과 공통

두 개 이상의 조건을 만족시키는 경우를 선택하고자 하는 경우 각각의 선택자를 모두 합친 것을 선택자로 사용하면 됩니다.
두 개 이상에 같은 종류의 CSS 코드를 적용하고 싶은 경우, 콤마(,)를 통해 구분하여 적으면 됩니다.

문법

선택자 지정: (TAG name).(class)#(id) { property1: value1; property2: value2; ...... } (각 요소 선택)

예시

몇몇 태그들: A. <p>......</p>
B. <p class="ABC">......</p>
C. <p class="DEF">......</p>
D. <p class="ABC" id="asdf">......</p>
E. <p class="DEF" id="asdf">......</p>
F. <h1 class="ABC">......</h1>

각 선택자에 대해 해당되는 태그들: p A, B, C, D, E .ABC B, D, F p.ABC B, D #asdf D, E .ABC#asdf D

ps. 여러 선택자들을 알고 싶으시다면 여러분이 갖고 계신 최고의 도구인 검색 엔진을 사용해보세요.
