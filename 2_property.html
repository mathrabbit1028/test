CSS 속성

CSS에서 속성은 변경할 수 있는 디자인 요소를 말합니다.
CSS 속성은 콜론(:)을 통해 지정하며, 지정되지 않은 경우 기본값으로 설정됩니다.
하나의 style에서 여러 CSS 속성을 변경할 수 있으며, 각각의 속성은 세미콜론(;)으로 구분되어야 합니다.

문법

style 속성에서의 CSS 코드: style="(property1): (value1); (property2): (value2); ......"

예시

WOW : <span style="color:red;">WOW</span> WOW : <span style="text-align:center; text-decoration: line-through;">WOW</span>

ps. 여러 속성들을 알고 싶으시다면 여러분이 갖고 계신 최고의 도구인 검색 엔진을 사용해보세요.
